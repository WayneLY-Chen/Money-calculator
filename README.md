> **「優雅地處理每一分花費。」** > Money-Calculator 是一個基於 Python 的財務邏輯引擎，專為解決團體聚餐中「消費不均」的痛點而設計。

## :book: 核心理念

在真實社交場景中，分帳往往不只是「總金額除以人數」。有人點了昂貴的調酒，有人只喝了水；ElegantSplit 的存在就是為了確保每一位參與者只需為其「實際消費」與「公用項目」買單。

## :sparkles: 關鍵特性

- **精準歸屬邏輯**：支援從總帳中抽離特定費用（如酒水），僅由參與者分攤。
- **動態權重計算**：自動處理公用項目與個人加選項目的加總，消除手算誤差。
- **高擴展性**：模組化設計，可輕鬆對接 FastAPI 或 React Native 移動端介面。
- **非對稱處理**：完美應對「有人有點、有人沒點」的複雜分帳需求。

### 1. 環境需求

確保您的系統已安裝 Python 3.8 或以上版本。
